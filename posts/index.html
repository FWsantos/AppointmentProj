Instalação

written by FWsantos on 2016-12-29

Para poder instalar a versão Appointment com Exprees, é necessário ter NodeJS na sua máquina. Os pacotes para instalação em diferentes sistemas operacionais estão disponíveis em https://nodejs.org/en/download/ .

Já com Node na sua máquina:

  1. Baixe o arquivo zip ou dê git clone no diretório AppointmentProj.

  2. Acesse o branch 'A_ZONA' : git checkout A_ZONA.

  3. Abra o terminal, acesse o diretório e dentro dele digite o comando npm install. Ele irá instalar as dependências.

  4. Agora vamos testá-lo em localhost. Para isso vamos usar o nodemon. Se não tiver ele instale com o comando npm install nodemon. Quando terminar instalação ele irá apresentar a árvore de pastas.

  5. Digite nodemon. Depois que aparecer [nodemon] starting `node ./bin/www`, acesse o navegador com a url http://localhost:3000/.
